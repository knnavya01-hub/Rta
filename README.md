# Ṛta - Reflective Intelligence

> *ṛta* (ऋत) - one of the oldest recorded words in human language. The natural order beneath all things.

**Order. Alignment. Balance.**

Ṛta is an AI-powered clarity engine for emotional and cognitive load. It listens to what is weighing on you, finds the wisdom within it, and returns it as a cinematic experience - ancient scripture, animated into light.

🌐 **Live at [rta-app.netlify.app](https://rta-ai.netlify.app)**

---

## What it does

You type or speak what is on your mind. Ṛta senses the emotional weight beneath your words, matches it to the right wisdom using a custom retrieval pipeline, generates six watercolor scenes as an animated visual story, and returns clarity - not advice.

Not a chatbot. Not a journaling app. A reflective intelligence.

---

## This repository

This repo contains the **frontend only** - the PWA interface.

```
frontend/
  index.html      - full app shell
  manifest.json   - PWA manifest
  sw.js           - service worker
```

### Frontend
- Vanilla HTML · CSS · JavaScript
- Canvas API - twinkling starfield, nebula blobs, constellation lines, meteors
- Web Speech API - voice input
- SVG animations - neon wire icon sparks, glowing clarity ring
- PWA - installable, works offline

---

## Backend - Private

The backend is proprietary and lives in a private repository.

It includes a custom emotional sensing layer, a retrieval pipeline across ancient scripture, a multi-stage AI generation system, session memory, and crisis support integrations.

Available for review upon request.

---

## Installing as a PWA

Once live on HTTPS:

- **Android** - Chrome → three dots → Add to Home Screen
- **iOS** - Safari → Share → Add to Home Screen

Installs like a native app. No app store needed.
